\subsection{Statistiques descriptives -- variables quantitatives}

La seule variable quantitative retenue est \textbf{V2 : Annales travaillées}
(variable quantitative discrète, Déf.~1.3 ; la modalité \og{}4+\fg{} est recodée
en~4). Les paramètres suivants sont calculés sur \(n = 68\) réponses valides.

\noindent\textbf{Calculs détaillés.}
\begin{itemize}
  \item \(\bar{x} = \tfrac{1}{68}\sum x_i \approx 2{,}1912\) \emph{(Déf.~1.4)}
  \item \(M_e\) : \(n = 68\) pair \(\Rightarrow M_e = \tfrac{x_{(34)} + x_{(35)}}{2} = \tfrac{2+2}{2} = 2\)
        \emph{(Déf.~1.5, PM~1.1)}
  \item \(Q_1\) : \(n \equiv 0 \pmod{4}\) \(\Rightarrow Q_1 = \tfrac{x_{(17)} + x_{(18)}}{2} = \tfrac{2+2}{2} = 2\)
        \emph{(Déf.~1.6, PM~1.2)}
  \item \(Q_3 = \tfrac{x_{(51)} + x_{(52)}}{2} = \tfrac{3+3}{2} = 3\)
  \item \(\sigma^2 = \overline{x^2} - \bar{x}^2 = \tfrac{1}{68}\sum x_i^2 - (2{,}1912)^2 \approx 0{,}7723\)
        \emph{(Déf.~1.9)}
  \item \(S_e^2 = \tfrac{n}{n-1}\,\sigma^2 = \tfrac{68}{67}\times 0{,}7723 \approx 0{,}7838\)
        \emph{(§\,2.3.1)}
\end{itemize}

\paragraph{Corrélation avec la performance académique (Déf.~1.16).}

On calcule le coefficient de corrélation linéaire entre V2 et V12
(note de bac, recodée numériquement 2--5, sur \(n = 67\) paires valides) :
\[
  r(V2,\,V12)
  = \frac{\operatorname{Cov}(V2,\,V12)}{\sigma_{V2}\times\sigma_{V12}}
  \approx 0{,}045
\]
\(\lvert r \rvert \approx 0\) : \textbf{absence de corrélation linéaire}
entre le nombre d'annales travaillées et la note de bac (Déf.~1.16).
